# Spring Framework Apps - Two Projects (Task1 + Task2)

This repository contains two separate Spring Boot applications:

- `Task2FirstRestAPI` - REST API + Swagger + H2 + Spring Data JPA

Open each folder as a separate Maven project in IntelliJ.
